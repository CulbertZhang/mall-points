# 商城会员积分系统

基于 Vue 3 + Express + PostgreSQL 的全栈商城会员积分系统。

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite + Vant 4 + Pinia
- **后端**: Express + TypeScript + Prisma ORM
- **数据库**: PostgreSQL + Redis

## 快速开始

```bash
# 安装依赖
npm install

# 启动后端
npm run dev:backend

# 启动前端
npm run dev:frontend
```

## 项目结构

```
packages/
├── frontend/    # Vue 3 前端
└── backend/     # Express 后端
```
